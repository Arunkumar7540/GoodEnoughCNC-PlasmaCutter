AFFORDABLE & USEFUL open source CNC machines for
designers, engineers and all other creators

Prototyping and production

GoodEnoughCNC family facilitates the creation by lowering the costs of production. Designers and engineers use the machines for idea testing, prototyping and for small series production.

Education

GoodEnoughCNC machines are used in education. Students learn hands on by building the machines. As well, they are empowered to realize their own projects at lows cost using the GoodEnoughCNC machine toolset.

DIY making

Perfect for local maker spaces with limited resources, where community can use GoodEnoughCNC machine tools for their DIY make and repair projects.

Build your own GoodEnoughCNC!

DIY VERSION

A DIY friendly open-source release as well as detailed instructions on how to properly assemble and test the system is available for GoodEnoughCNC Plasma Cutter. Design is suitable for almost everyone since almost no prior experience in building is needed. Build your own GoodEnoughCNC machine in just a few days with only basic tools found in any garage.

A DIY friendly open-source release for other GoodEnoughCNC machine tools will be available soon.
