Precise XY

You can find documentation files for Precise XY, affordable and precise XY and Z movement platform for scientific experimentation in our Github repository PreciseXY. A DIY friendly open-source release, with KITs of the system available along with assembly instructions, will be avaliable shortly. For more information and enquiries pleaseÂ contact us or sign up for release news!

We have observed a lack of a low-cost and simple to construct 3 axis linear translation systems for use in scientific and research applications, ranging from laser beam profiling to automation in plant sciences. PreciseXY project aim to introduce a low-cost and yet precise and stable translation system ready for integration in any project with minimal work.
